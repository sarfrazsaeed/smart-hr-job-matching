How to Use This System
  • Open the Candidate page and register candidate details.
  • Navigate to the HR page to post job requirements.
  • Go to the Match page and click "Match Candidates".
  • The system will display candidates that match job criteria.
Project Information

Project Type: Academic Web Project
Technologies Used: HTML, CSS, Bootstrap, JavaScript
Data Storage: Browser localStorage

This system is designed for learning purposes and demonstrates basic job-candidate matching logic using client-side technologies.
