About This Project

Candidate HR Match About Help

The Smart HR & Job Matching System is a web-based application developed using HTML, CSS, Bootstrap, and JavaScript.

This system allows candidates to register their personal details, skills, education, and experience.

HR managers can post job openings with required skills, experience, and job type.

The matching module compares candidate profiles with job requirements and displays suitable matches.

All data is stored using browser localStorage, making the system fast, simple, and suitable for academic projects.
