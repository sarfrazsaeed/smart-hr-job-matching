SmartHire — HR & Job Matching System

A web-based HR management tool built with HTML, CSS, Bootstrap, 
and JavaScript. It allows candidate registration, job posting, 
and skill-based matching with percentage scores.

Live Demo
[View Live Project](https://sarfrazsaeed.github.io/smart-hr-job-matching)

 Features
•	Candidate registration with duplicate email prevention
•	HR job posting portal with job type selection
•	Skill-based matching with percentage score 0 to 100
•	Ranked results — best match shown first
•	Match score badges green yellow red
•	Dashboard with total stats and skill distribution chart
•	Toast notifications instead of browser alerts
•	Fully responsive works on mobile and desktop

Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript Vanilla
- Browser localStorage

Pages
| Page | Description |
| index.html | Candidate registration form |
| hr.html | HR job posting portal |
| match.html | Job candidate skill matching |
| dashboard.html | Stats and skill distribution |
| about.html | Project information |

How to Run
1. Clone or download this repository
2. Open index.html in any browser
3. No installation or server required
